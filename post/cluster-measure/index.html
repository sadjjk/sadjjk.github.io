聚类算法效果衡量标准

2019-06-29 Sat Work

聚类算法中常用的评估指标 确定最优k值

聚类效果指标分为无监督指标(实际类别信息未知)和有监督指标(实际类别信息已知)。

没有一个指标是一招鲜 通常结合多个指标 来判断最佳K值

无监督指标：实际类别信息未知

肘部法

SSE：误差平方和

每个样本点到其所属簇心的距离的平方和
因此如果每个点自成一簇，则SSE为0
理论上簇(K值)越多，SSE越小
在这个平方和变化过程中，会出现一个拐点，即“肘”点，下降率突然变缓 时即认为是最佳的k值。

import numpy as np
from sklearn.cluster import KMeans
from scipy.spatial.distance import cdist

kmeans = KMeans(4)
kmeans.fit(X)

cdist(X, kmeans.cluster_centers_, 'euclidean') # 每个样本到各个簇心的距离
np.min(cdist(X, kmeans.cluster_centers_, 'euclidean'),axis=1) # 每个样本点到所属簇心的距离，即距离最短
SSE = sum(np.min(cdist(X, kmeans.cluster_centers_, 'euclidean'), axis=1)) # SSE值

SC：轮廓系数法

SC值越大越好 [-1,1]

a是Xi与同簇的其他样本的平均距离，称为凝聚度；
b是Xi与最近簇中所有样本的平均距离，称为分离度。
求出所有样本的轮廓系数后再求平均值就得到了平均轮廓系数。
平均轮廓系数的取值范围为[-1,1]，系数越大，聚类效果越好。

import numpy as np
from sklearn.cluster import KMeans

kmeans_model = KMeans(n_clusters=3, random_state=1).fit(X)
labels = kmeans_model.labels_

metrics.silhouette_score(X, labels, metric='euclidean')

CH：Calinski-Harabasz

CH值越大越好

from sklearn import metrics
metrics.calinski_harabaz_score(X, labels)

有监督指标：实际类别信息已知

调整兰德系数 ARI

公式怎么算的没看懂

&&&&&UPDATE 2019年07月01日23:43:41&&&&&

终于搞明白了 是怎么计算的

$ARI(A,B) = \frac{\sum\limits_{i = 1}^{K_A}\sum\limits_{j = 1}^{K_B} \left(\begin{array}{c}2\ {N_{ij}}\end{array}\right) -t3 } {\frac{t_1+t_2}{2} - t3} $

其中，A和B是数据集Z的真实值和预测值的划分，分别有K~A~和K~B~个簇；N~ij~表示在划分A的第i个簇中的数据同时也在划分B的第j个簇中数据的数量；N~i~、N~j~分别表示划分A中第i个簇与划分B的第j个簇中的数据质量
